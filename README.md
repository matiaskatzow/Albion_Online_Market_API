# Albion_Online_Market_API guide

# Cómo ejecutar este proyecto
1. Clonar el repositorio o descargar los archivos.
2. Instalar las dependencias:
   pip install -r requirements.txt (si no funciona, intentar con py m- pip install -r requirements.txt)(nota, puede que no funcione)
3. Ejecutar la API:
   python market_albion_api.py
4. Haga click en "try it out y añada en "item name" el tier y el nombre del objeto deseado
5. Disfrutar
